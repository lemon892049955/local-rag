# RAG 评测测试集

> 用于评估 Local RAG 检索质量和答案生成能力。
> 每个测试用例包含：问题、期望答案关键词、期望来源、难度等级

---

## 测试集说明

| 维度 | 说明 |
|------|------|
| **精确查找** | 明确指向某篇文章/概念，测试检索命中率 |
| **细节提取** | 需要从文档中提取具体事实/数据/步骤 |
| **主题概览** | 宏观了解某个主题，测试综合能力 |
| **跨文档关联** | 需要综合多篇文档回答 |
| **否定测试** | 知识库中不存在的内容 |

## 评测指标

| 指标 | 计算方式 | 目标值 |
|------|----------|--------|
| **检索命中率** | 期望来源在 Top-3 中 | ≥ 80% |
| **答案准确率** | 包含期望关键词 | ≥ 70% |
| **否定识别率** | 正确识别知识库外内容 | ≥ 90% |
| **跨文档关联率** | 综合多篇文档回答 | ≥ 60% |
