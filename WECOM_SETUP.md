# 企业微信接入指南 - "转发即入库"

## 最终效果

手机/PC 刷到好文章 → 转发给「知识库助手」→ 1 秒回复"已收到" → 后台自动入库 → 推送"✅入库成功"卡片

直接发文字 → 自动搜索知识库 → 返回 RAG 答案

---

## 第一步：注册企业微信 (5 分钟)

1. 打开 [https://work.weixin.qq.com](https://work.weixin.qq.com)
2. 点击"立即注册" → 选择"企业" → 填写信息
   - 企业名称随便填，比如"我的知识库"
   - 行业选"IT/互联网"
   - 人员规模选最小的
3. 注册完成后进入管理后台

## 第二步：创建自建应用 (3 分钟)

1. 进入管理后台 → **应用管理** → 自建 → **创建应用**
2. 填写：
   - 应用名称：`知识库助手`（这个名字会显示在你微信通讯录里）
   - 应用 Logo：随便上传一个图标
   - 可见范围：选择你自己
3. 创建完成后，记录下：
   - **AgentID**：应用详情页顶部
   - **Secret**：点击"查看"获取（注意保管）

## 第三步：获取企业 ID

1. 管理后台 → **我的企业** → 企业信息
2. 最底部有 **企业ID** (CorpID)，记录下来

## 第四步：配置接收消息回调

1. 进入你创建的应用 → **接收消息** → 设置 API 接收
2. 填写：
   - **URL**：`https://你的域名/wecom/callback`
   - **Token**：随机字符串（比如 `MyKnowledgeBot2026`）
   - **EncodingAESKey**：点"随机获取"自动生成 43 位密钥
3. 先不要点保存！需要先启动你的服务。

> ⚠️ **URL 必须是公网可访问的 HTTPS 地址**
>
> 如果你的服务在本地，需要用内网穿透工具：
> - [ngrok](https://ngrok.com)：`ngrok http 8900` 获取临时公网地址
> - [frp](https://github.com/fatedier/frp)：自建稳定隧道
> - 云服务器：直接部署最稳定

## 第五步：配置环境变量

编辑项目的 `.env` 文件，填入上面获取的信息：

```bash
# 企业微信配置
WECOM_CORP_ID=ww1234567890abcdef      # 第三步获取的企业 ID
WECOM_SECRET=xxxxxxxxxxxxxxxxxxxx      # 第二步获取的应用 Secret
WECOM_AGENT_ID=1000002                 # 第二步获取的 AgentID
WECOM_TOKEN=MyKnowledgeBot2026         # 第四步你设置的 Token
WECOM_ENCODING_AES_KEY=xxxxxxxxxxx...  # 第四步的 EncodingAESKey (43位)
```

## 第六步：启动服务 & 验证

```bash
# 启动服务
python cli.py serve

# 如果需要内网穿透 (另一个终端)
ngrok http 8900
# 复制 ngrok 给你的 https://xxx.ngrok.io 地址
```

回到企微后台，将回调 URL 填为 `https://xxx.ngrok.io/wecom/callback`，点击保存。
如果显示"验证成功"，说明服务已对接成功！

## 第七步：开启微信插件

1. 管理后台 → **我的企业** → **微信插件**
2. 用你的**个人微信**扫描页面上的二维码
3. 关注后，你的微信通讯录里会出现「知识库助手」

## 开始使用！

### 入库
微信里看到好文章 → 点"转发" → 搜索「知识库助手」→ 发送

系统会：
1. 立即回复 "📥 已收到，正在处理中..."
2. 后台自动抓取 → LLM 清洗 → Markdown 落库 → 向量索引
3. 完成后推送 "✅ 入库成功" 卡片（含标题、标签）

### 搜索
直接给「知识库助手」发文字，比如：
> "之前那篇关于 Vibe Coding 的文章提到了哪些步骤？"

系统会搜索知识库并返回 RAG 答案。

                          ┌──────┬───────┼───────┬──────┐
                          ▼      ▼       ▼       ▼      ▼

## 文件说明

```
wecom/
├── __init__.py
├── crypto.py      # 企微消息加解密 (AES + 签名)
├── sender.py      # 主动推送消息 (access_token + 发消息API)
└── callback.py    # 回调路由 (GET验证 + POST接收 + 异步入库)
```
