## v2.0 (2026-04-11) — RAG 检索优化

### 核心改动
- **意图路由**：新增 `_classify_intent()` 规则优先 6 类意图分类（detail/precise/overview/entity/fuzzy/negative）
- **分路加权**：`INTENT_WEIGHTS` 不同意图对 data/topics/concepts/entities/moc 差异化权重
- **搜索流程重构**：粗排 Top20 → 分路加权 → Reranker 精排 → Top3 裁剪 → LLM 生成
- **洞察质量修复**：增强空洞察过滤（8 种模式匹配）、去重阈值 0.80→0.75、清理代码块包裹、去除重复来源引用

### 改动文件
- `retrieval/hybrid_searcher.py` — 意图分类 + 分路加权 + Top3 裁剪
- `wiki/compiler.py` — 洞察过滤增强 + 去重收紧
- `wiki/page_store.py` — 追加块去除重复引用

## v1.0 (2026-04-11) — Wiki 知识层重构

- `wiki/moc_builder.py` — 新建，MOC 导航页生成器

## v0.9.3 (2026-04-10) — LLM 切换 + Prompt 强化

### 核心改动
- **LLM 切换**：Kimi → DeepSeek (`sk-c3e1...`)
- **RAG Prompt 6 条铁律**：无文档不回答/不编造/抗诱导/简洁切题/来源标注/矛盾标记
- **Cross-Encoder Reranker**：`bge-reranker-v2-m3` 精排
- **HTTPS 部署**：域名 `buddyknow.top` + Caddy 自动证书

### 数据结果
- Recall 78.3% → 81.9% | Hit Rate 87.0% → 91.3% | 否定测试 → 100% | 零错误

## v0.9 (2026-04-10) — 检索 P0 优化

- Recall 41.3% → 78.3% (+37pp) | Hit Rate 50.0% → 87.0% (+37pp)

## v0.8 (2026-04-10) — Wiki 编译质量优化

### 三轮迭代（v0.3→v0.7→v0.8）
