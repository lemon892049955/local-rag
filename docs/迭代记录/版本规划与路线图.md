
## 三、后续待做（P1/P2）

### P1-1：模糊查询优化
- 多 Query 改写 + 标题加权
- 预期：模糊查询 Hit Rate 60% → 80%+

### P1-2：SSE 流式输出 + 链路并行
- 搜索接口改 SSE，首字 2-3s 返回
- 查询改写与检索并行
- 预期：体感延迟 → 3s

### P2：质量收尾
- Keyword Score 优化（Prompt 微调"简洁但完整"）
- 概念卡抽取精度提升（Precision 28%→80%）
- Gate 验收通关（阶段一剩余 2 项 + 阶段二全项）
- Few-shot 反馈闭环（企微 👍👎 → 好答案做示例）
