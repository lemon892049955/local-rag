# Anthropic的Claude Code的Auto Dream功能

> Anthropic的Claude Code新增Auto Dream功能，使得Agent在休息时能够自动生成梦境。

## 核心内容

### 功能介绍
Anthropic最近推出了一个名为Auto Dream的新功能，该功能使得Agent在休息时能够自动生成梦境。

### 功能特点
- **自动生成梦境**：Claude Code在用户不使用时，会在后台自动生成梦境。
- **灰度测试**：该功能从3月底开始灰度测试，目前尚未全面开放。
- **趣味性与实用性**：作者认为这个功能既有趣又实用，并决定分享给更多人。
