# 深度解析GitHub上的'同事.skill'项目及其Agent_Skills架构

> 探索GitHub上'同事.skill'项目的技术架构和Agent Skills的发展现状。

## 核心内容

### 项目概述
2026年4月，GitHub上的“同事.skill”项目在5天内获得了6600+ Stars，引发全网热议。该项目通过提供同事的邮件聊天记录、钉钉文件和工作文档，AI能生成一个完美复刻其工作方式的数字分身。

### 技术深度解析：SKILL.md 同层架构
Agent Skills是2025年由Anthropic率先发布的AI能力扩展标准。核心思想是：一个SKILL.md文件就能给AI装上新能力。2026年，该标准被Cursor、Codex CLI、OpenClaw全面采用，Skill Marketplace已收录70万+技能包。

### Agent Skills 生态全貌：70万+技能包的世界
这些爆火的“人格.skill”都采用了相同的架构，包括知识层（Memory）和人格层（Persona）。

### 踩坑记录与最佳实践
随种保护示例：

```python
import re

def sanitize_skill_data(content:str)->str:
    ""“发布Skill前清除敏感信息”""
    patterns={'phone':r'1[3-9]\d{9}','email':r'[\w.-]+@[\w.-]+\.\w+','id_card':r'\d{17}[\dXx]','name':r'(?:张|李|王|刘|陈)\w{1,2}(?=\s|,|。|：)',}
    for pii_type,pattern in patterns.items():
        content=re.sub(pattern,f'[{pii_type}_REDACTED]',content)
    return content

raw_skill=open("colleague_memory.md").read()
safe_skill=sanitize_skill_data(raw_skill)
open("colleague_memory.md","w").write(safe_skill)
```
