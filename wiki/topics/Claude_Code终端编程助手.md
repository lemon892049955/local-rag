# Claude Code：终端中的编程助手

> Claude Code是一个终端编程工具，通过自然语言命令帮助执行常规任务、解释复杂代码和处理Git工作流。

## 核心内容

### 功能概述
Claude Code是一个代理编程工具，它存在于终端中，理解你的代码库，并帮助你通过执行常规任务、解释复杂代码和处理Git工作流来更快地编码，所有这些都可以通过自然语言命令完成。它可以在你的终端、IDE或通过在GitHub上标记@claude来使用。

### 安装方法
- **MacOS/Linux (推荐)**:
  ```
  curl -fsSL https://claude.ai/install.sh|bash
  ```
  或使用Homebrew:
  ```
  brew install --cask claude-code
  ```
- **Windows (推荐)**:
  ```
  irm https://claude.ai/install.ps1|iex
  ```
  或使用WinGet:
  ```
  winget install Anthropic.ClaudeCode
  ```
- **NPM (已弃用)**:
  ```
  npm install -g @anthropic-ai/claude-code
  ```

### 使用方法
导航到你的项目目录并运行`claude`。

### 插件与功能扩展
这个仓库包括几个Claude Code插件，它们通过自定义命令和代理扩展功能。查看插件目录以获取有关可用插件的详细文档。

### 反馈与社区
我们欢迎你的反馈。使用`/bug`命令在Claude Code内直接报告问题，或在GitHub上提出问题。

### 数据收集、使用和保留
使用Claude Code时，我们会收集反馈，包括使用数据（例如代码接受或拒绝）、相关对话数据和通过`/bug`命令提交的用户反馈。我们实施了几项保护措施来保护你的数据，包括对敏感信息的有限保留期、对用户会话数据的限制访问和明确政策反对使用反馈进行模型训练。
