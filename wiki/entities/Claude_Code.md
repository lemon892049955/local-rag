# Claude Code：终端中的编程助手

> Claude Code是一个终端编程工具，通过自然语言命令帮助执行常规任务、解释复杂代码和处理Git工作流。

## 基本信息

- **功能**: 执行常规任务、解释复杂代码、处理Git工作流
- **使用场景**: 终端、IDE、GitHub
- **安装方法**: MacOS/Linux推荐使用curl或Homebrew，Windows推荐使用PowerShell脚本或WinGet，NPM安装方法已弃用
- **插件支持**: 支持自定义命令和代理的插件
- **社区**: Claude Developers Discord社区
- **数据政策**: 收集反馈、使用数据和保留政策，包括敏感信息的有限保留期、用户会话数据的受限访问和反对使用反馈进行模型训练的政策
